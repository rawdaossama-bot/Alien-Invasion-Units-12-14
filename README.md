# Alien Invasion

**Author:** Rawda Hassanin  
**Date:** 11/30/2025  

## Project Description
Alien Invasion is a 2D space shooter game built using Python and Pygame.  
The player controls a spaceship that moves horizontally at the bottom of the screen and fires bullets to defend against enemies (or simply practice shooting mechanics).  
This project demonstrates OOP concepts, modular design, Pygame functionality, and professional code documentation.

---

## How to Run

### Requirements
- Python 3.10+ (or your installed version)
- Pygame library

### Installation
1. Clone the repository:
```bash
git clone https://github.com/rawdaossama-bot/Lab12_Rawda-Hassanin_1.git

cd Lab12_Rawda-Hassanin_1

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   pip install pygame
   ```

Run the game:
python alien_invasion.py


How to Play

Move ship left: Left Arrow

Move ship right: Right Arrow

Fire bullets: Spacebar

Quit game: Q
